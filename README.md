# BOTCADE

Single-player AI arcade platform with strategy and casual games against intelligent bots.

## Games

| Game | AI Algorithm |
|------|-------------|
| XO (Tic Tac Toe) | Minimax |
| Connect 4 | Alpha-Beta Pruning |
| Memory Match | Probabilistic Simulation |
| Sudoku | Backtracking Solver |
| Chess | Minimax Depth 3 |

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
